A Hustle Tech começou como um portfólio de projetos pessoais e virou algo que se paga. Aqui vai a versão honesta.

## A stack

Nada exótico, tudo deliberado:

- **Backend:** microsserviços Java + Spring Boot na GCP
- **Frontend / mobile:** React, React Native
- **Dados:** PostgreSQL, Redis
- **Infra:** Terraform, CI/CD com GitHub Actions

A restrição que moldou cada decisão: *eu tinha que conseguir operar isso sozinho, à 1h da manhã, depois de um dia inteiro de trabalho para clientes.*

## Agentes como multiplicador de força

Internamente, me apoio em agentes LLM para testes automatizados, revisão de arquitetura, varredura de vulnerabilidades e enforcement de qualidade de código. O tempo de code review saiu de gargalo para não-evento.

> Não siga tendências. Crie-as.

## O que aprendi

Lucratividade é uma função forçante. Me deixou implacável com escopo e generoso com confiabilidade, porque cada cliente pagante é uma relação, não uma métrica.
